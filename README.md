# Tic-Tac-Toe

A polished browser-based Tic-Tac-Toe game with a responsive board, score tracking, and AI mode.

## Features

- Two-player mode
- Play vs AI mode
- Scoreboard for X, O, and draws
- Responsive layout with side-by-side game and dashboard panels
- Animated board interactions

## Files

- `index.html` — main page markup
- `styles.css` — visual layout and styling
- `script.js` — game behavior and logic

## Usage

1. Open `index.html` in a browser.
2. Click a cell to place your move.
3. Use the mode buttons to switch between Two-player and Play vs AI.
4. Click `Restart Game` to reset the board.

## Notes

- The UI uses a two-panel layout on wider screens and stacks vertically on smaller screens.
- Audio cues are included for moves, wins, draws, and mode changes.
